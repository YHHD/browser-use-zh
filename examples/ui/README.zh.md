# **Browser-Use 的用户界面**

| **文件名**            | **用户界面** | **描述**                           | **示例用法**                      |
|------------------------|-------------------|-------------------------------------------|-------------------------------------------|
| `command_line.py`      | **终端**      | 解析用于命令行执行的参数。| `python command_line.py`                  |
| `gradio_demo.py`       | **Gradio**        | 提供基于 Gradio 的交互式 UI。  | `python gradio_demo.py`                   |
| `streamlit_demo.py`    | **Streamlit**     | 运行基于 Streamlit 的网络界面。    | `python -m streamlit run streamlit_demo.py` |
